# 特徴 

  * フォロー、フォロワー機能で、各ユーザーの信用度が可視化出来ます。
  * ユーザーごとに店舗情報の作成が出来る為、来店に繋げられるような仕組みになっています。

# 作成意図
  * 前職でドラッグストア業界に勤務していた時、店舗の接客以外でお客様の役に立ち、店舗従業員の評価が可視化され、モチベーションが上がるような仕組みが出来ないか考え、作成致しました。
